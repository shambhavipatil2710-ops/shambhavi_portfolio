# Shambhavi - Java Full Stack Developer Portfolio

Welcome to the repository for my personal developer portfolio! This is a modern, premium, and fully responsive personal website built to showcase my skills, experience, and projects.

## 🎨 Theme & Design
The portfolio features a custom **Bidar Fort Dark Theme**. It seamlessly blends historical Indian architecture with high-tech modern web design:
- **Color Palette:** Deep brown and black (`#060402`, `#1a1007`) paired with luxurious gold and amber accents (`#e5b95f`, `#f5e6c8`).
- **Aesthetics:** Glassmorphism UI cards, glowing neon elements, and sleek typography (`Samsung Sharp Sans`, `Oswald`, `Space Mono`, and `Dancing Script`).
- **Custom Preloader:** A 5-second premium loading screen featuring a glowing code interface, dynamic progress bar, and tech icons.
- **3D Animations:** Scroll-driven 3D card tilts, micro-interactions, and floating particles that bring the interface to life.

## 🛠️ Technologies Used
- **HTML5**: Semantic markup for content structure.
- **CSS3**: Vanilla CSS with custom properties (CSS variables), Flexbox, CSS Grid, complex keyframe animations, and glassmorphism.
- **JavaScript**: Vanilla JS for DOM manipulation, preloader logic, scroll-driven animations, and mobile navigation overlay.
- **Icons**: FontAwesome and Devicon for sleek social and technology stack icons.

## 🚀 Features
- **Immersive Loading Screen:** Custom glowing loader with a simulated boot-up sequence and background code snippets.
- **Hero Section:** Features a dynamic 3D-tilt profile card, animated background clouds, and glowing orbs.
- **About & Skills:** Visually engaging skill grids highlighting expertise in Java, Spring Boot, React, IoT, and Data Science.
- **Experience & Projects:** Structured timelines and interactive project cards showcasing my journey and work.
- **Fully Responsive:** Beautifully optimized for desktop, tablet, and mobile devices with a custom hamburger menu overlay.

## 💻 How to Run Locally
Since this is a static website built with pure HTML/CSS/JS, no complex build tools or servers are required!
1. Clone the repository or download the folder.
2. Open `index.html` directly in your favorite web browser (Chrome, Firefox, Edge, Safari).
3. Alternatively, you can use a local development server like VS Code's "Live Server" extension for a better development experience.
